Guía de Diseño Atómico – Sistema de Componentes

Este documento describe la estructura de componentes basada en la metodología Atomic Design.
Cada componente incluye su categoría (Átomo, Molécula), nombre estándar, descripción y lista de props.

1. Átomos

1.1 TextInput

Descripción: Campo de entrada de texto estándar.

Props:

  • value: string – Texto actual.
  • onChange: (value: string) => void – Callback al cambiar el valor.
  • placeholder?: string – Texto placeholder.
  • disabled?: boolean – Deshabilita el campo.
  • error?: string – Mensaje de error.
  • size?: 'sm' | 'md' | 'lg' – Tamaño del campo.
  • iconLeft?: ReactNode – Icono al inicio.
  • iconRight?: ReactNode – Icono al final.

1.2 PasswordInput

Descripción: Campo para contraseñas con opción de mostrar/ocultar.

Props:

  • value: string
  • onChange: (value: string) => void
  • placeholder?: string
  • disabled?: boolean
  • error?: string
  • size?: 'sm' | 'md' | 'lg'
  • showToggle?: boolean – Mostrar botón para alternar visibilidad.

1.3 EmailInput

Descripción: Campo optimizado para correos electrónicos.

Props:

  • value: string
  • onChange: (value: string) => void
  • placeholder?: string
  • disabled?: boolean
  • error?: string

1.4 NumericBadge

Descripción: Etiqueta numérica para notificaciones o contadores.

Props:

  • count: number
  • max?: number – Número máximo mostrado antes de usar formato +99.
  • color?: string
  • size?: 'sm' | 'md' | 'lg'

1.5 Button

Descripción: Botón de acción principal o secundaria.

Props:

  • label: string
  • onClick: () => void
  • variant?: 'primary' | 'secondary' | 'tertiary' | 'danger'
  • size?: 'sm' | 'md' | 'lg'
  • iconLeft?: ReactNode
  • iconRight?: ReactNode
  • disabled?: boolean
  • fullWidth?: boolean

1.6 IconButton

Descripción: Botón circular o cuadrado que muestra solo un icono.

Props:

  • icon: ReactNode
  • onClick: () => void
  • variant?: 'filled' | 'outline' | 'ghost'
  • size?: 'xs' | 'sm' | 'md' | 'lg'
  • disabled?: boolean
  • ariaLabel: string – Accesibilidad.

1.7 IconToggle

Descripción: Botón que alterna entre dos estados (ej. favorito).

Props:

  • active: boolean
  • onToggle: (active: boolean) => void
  • iconActive: ReactNode
  • iconInactive: ReactNode
  • size?: 'sm' | 'md' | 'lg'
  • ariaLabel: string

1.8 Checkbox

Descripción: Casilla de verificación.

Props:

  • checked: boolean
  • onChange: (checked: boolean) => void
  • label?: string
  • disabled?: boolean

1.9 RadioButton

Descripción: Opción de selección única.

Props:

  • checked: boolean
  • onChange: () => void
  • label?: string
  • name: string
  • value: string
  • disabled?: boolean

1.10 Switch

Descripción: Control on/off tipo interruptor.

Props:

  • checked: boolean
  • onChange: (checked: boolean) => void
  • size?: 'sm' | 'md' | 'lg'
  • disabled?: boolean

1.11 Tag

Descripción: Etiqueta pequeña con texto o icono.

Props:

  • label: string
  • variant?: 'default' | 'success' | 'warning' | 'error'
  • iconLeft?: ReactNode
  • iconRight?: ReactNode
  • removable?: boolean
  • onRemove?: () => void

1.12 Avatar

Descripción: Imagen de usuario u objeto.

Props:

  • src: string
  • alt: string
  • size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl'
  • statusDot?: 'online' | 'offline' | 'busy'
  • onClick?: () => void

1.13 SearchInput

Descripción: Campo de búsqueda con icono de lupa integrado.

Props:

  • value: string
  • onChange: (value: string) => void
  • placeholder?: string
  • disabled?: boolean
  • onClear?: () => void

1.14 Dropdown

Descripción: Lista desplegable básica.

Props:

  • options: { label: string; value: string }[]
  • value: string
  • onChange: (value: string) => void
  • placeholder?: string
  • disabled?: boolean

1.15 Tooltip

Descripción: Mensaje flotante para información contextual.

Props:

  • content: string | ReactNode
  • position?: 'top' | 'bottom' | 'left' | 'right'
  • trigger: 'hover' | 'click'

1.16 ProgressBar

Descripción: Indicador visual de progreso lineal.

Props:

  • value: number – Porcentaje completado (0-100).
  • color?: string
  • showLabel?: boolean

1.17 TagStatus

Descripción: Etiqueta de estado con color y texto.

Props:

  • label: string
  • status: 'active' | 'inactive' | 'pending' | 'error'
  • size?: 'sm' | 'md' | 'lg'

2. Moléculas

2.1 SearchBar

Descripción: Barra de búsqueda compuesta por SearchInput y botón de acción.

Props:

  • value: string
  • onChange: (value: string) => void
  • onSearch: () => void
  • placeholder?: string
  • filters?: ReactNode

2.2 UserMenu

Descripción: Menú desplegable para opciones de usuario con avatar.

Props:

  • avatarSrc: string
  • userName: string
  • options: { label: string; onClick: () => void }[]

2.3 Card

Descripción: Contenedor con imagen, título, descripción y acciones.

Props:

  • imageSrc?: string
  • title: string
  • description?: string
  • tags?: string[]
  • onClick?: () => void
  • actions?: ReactNode

2.4 MediaCard

Descripción: Variante de Card para videos o imágenes con overlay.

Props:

  • mediaType: 'image' | 'video'
  • src: string
  • title: string
  • subtitle?: string
  • onClick?: () => void
  • overlayContent?: ReactNode

2.5 CourseCard

Descripción: Tarjeta de curso con estado de progreso.

Props:

  • title: string
  • progress: number
  • duration?: string
  • thumbnail: string
  • onClick: () => void

2.6 FileCard

Descripción: Tarjeta para representar archivos.

Props:

  • fileName: string
  • fileType: string
  • thumbnail?: string
  • onClick?: () => void
  • onFavoriteToggle?: () => void
  • isFavorite?: boolean

2.7 AvatarGroup

Descripción: Grupo de avatares en lista horizontal.

Props:

  • avatars: { src: string; alt: string }[]
  • maxVisible?: number
  • size?: 'sm' | 'md' | 'lg'

2.8 Pagination

Descripción: Control de paginación numérica.

Props:

  • currentPage: number
  • totalPages: number
  • onPageChange: (page: number) => void

2.9 Tabs

Descripción: Navegación por pestañas.

Props:

  • tabs: { label: string; value: string }[]
  • activeTab: string
  • onTabChange: (value: string) => void

2.10 SidebarMenu

Descripción: Menú lateral con ítems y secciones.

Props:

  • sections: { title?: string; items: { label: string; icon?: ReactNode; onClick: () => void }[] }[]
  • activeItem?: string

3. Organismos

3.1 CourseList

Descripción: Lista de cursos con filtrado y paginación.

Props:

  • courses: CourseCard[]
  • filters?: ReactNode
  • pagination?: ReactNode

3.2 FileGallery

Descripción: Galería de archivos con favoritos y filtros.

Props:

  • files: FileCard[]
  • onFilterChange?: () => void
  • onSearch?: (query: string) => void

3.3 DashboardHeader

Descripción: Encabezado con buscador, avatar de usuario y acciones rápidas.

Props:

  • onSearch: (query: string) => void
  • userMenu: ReactNode
  • actions?: ReactNode

3.4 CourseDashboard

Descripción: Vista de cursos recientes, filtrados y categorías.

Props:

  • recentCourses: CourseCard[]
  • categories: string[]
  • onCategorySelect: (category: string) => void

3.5 MediaManager

Descripción: Vista principal para gestionar imágenes y videos.

Props:

  • mediaFiles: FileCard[]
  • onAddMedia: () => void
  • onSearch: (query: string) => void
  • filters?: ReactNode

Guía de Diseño Atómico – Continuación

1. Átomos adicionales

1.18 Link

Descripción: Enlace de navegación o acción.

Props:

  • href: string
  • label: string
  • target?: '_self' | '_blank'
  • onClick?: () => void
  • disabled?: boolean

1.19 Loader

Descripción: Indicador de carga circular o lineal.

Props:

  • type?: 'spinner' | 'bar'
  • size?: 'sm' | 'md' | 'lg'
  • color?: string

1.20 StatusDot

Descripción: Indicador de estado pequeño (ej. online/offline).

Props:

  • status: 'online' | 'offline' | 'busy' | 'away'
  • color?: string
  • size?: 'xs' | 'sm'

1.21 Icon

Descripción: Elemento gráfico representando acciones o estados.

Props:

  • name: string – Identificador del icono.
  • size?: number
  • color?: string

2. Moléculas adicionales

2.11 Breadcrumb

Descripción: Navegación jerárquica en forma de migas de pan.

Props:

  • items: { label: string; href?: string; onClick?: () => void }[]
  • separator?: ReactNode

2.12 Stepper

Descripción: Indicador de pasos en un flujo.

Props:

  • steps: { label: string; completed?: boolean }[]
  • activeStep: number
  • onStepClick?: (index: number) => void

2.13 ChatBubble

Descripción: Burbuja de chat con texto y metadatos.

Props:

  • message: string
  • timestamp?: string
  • isOwn?: boolean
  • status?: 'sent' | 'delivered' | 'read'
  • avatarSrc?: string

2.14 ChatInput

Descripción: Barra de entrada para chat con acciones rápidas.

Props:

  • value: string
  • onChange: (value: string) => void
  • onSend: () => void
  • placeholder?: string
  • actions?: ReactNode

2.15 FilterBar

Descripción: Conjunto de filtros aplicables a una vista.

Props:

  • filters: { label: string; value: string }[]
  • activeFilter: string
  • onFilterChange: (value: string) => void

2.16 VideoPlayerControls

Descripción: Controles de un reproductor de video.

Props:

  • isPlaying: boolean
  • onPlayPause: () => void
  • onSeek: (time: number) => void
  • onVolumeChange: (volume: number) => void
  • onFullscreenToggle: () => void

2.17 ChapterList

Descripción: Lista de capítulos con miniatura y timestamp.

Props:

  • chapters: { title: string; timestamp: string; thumbnail?: string }[]
  • onChapterSelect: (index: number) => void
  • activeChapter?: number

2.18 EmojiPicker

Descripción: Selector de emojis categorizados.

Props:

  • categories: { name: string; emojis: string[] }[]
  • onSelect: (emoji: string) => void

2.19 GifPicker

Descripción: Selector de GIFs con búsqueda.

Props:

  • gifs: { url: string; title?: string }[]
  • onSelect: (gifUrl: string) => void
  • onSearch: (query: string) => void

3. Organismos adicionales

3.6 ChatWindow

Descripción: Composición de lista de mensajes y entrada de texto.

Props:

  • messages: ChatBubble[]
  • onSendMessage: (text: string) => void
  • onLoadMore?: () => void

3.7 VideoPlayer

Descripción: Composición del reproductor de video con controles y lista de capítulos.

Props:

  • src: string
  • controls: ReactNode
  • chapters?: ChapterList
  • subtitles?: string

3.8 DashboardLayout

Descripción: Distribución general de panel con navegación y contenido.

Props:

  • sidebar: ReactNode
  • header: ReactNode
  • content: ReactNode

3.9 SettingsPanel

Descripción: Vista de configuración con secciones editables.

Props:

  • sections: { title: string; content: ReactNode }[]

3.10 PartnerManagement

Descripción: Vista para gestión de socios, con tabla, filtros y detalles.

Props:

  • partners: any[]
  • onEdit: (partner: any) => void
  • onDelete: (id: string) => void
  • filters?: ReactNode
